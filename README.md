# Тестовое задание

#### Версии
```bash
    node v14.21.3
    npm 6.14.18
    react v17.0.1
```

Клонировать

```bash
  git clone https://github.com/VladKnyaz/testTaskHammerSys.git
```
Зайти в директорию

```bash
  cd testTaskHammerSys
```

Установка зависимостей
```bash
  npm install
```

Сбилдить

```bash
  npm run build
```
